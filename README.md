# FINAL REPORT - Scientific Communication, Collaboration and Design in Quantitative Research Using R
A collaborative final report illustrating applied knowlegde from the course HEV9009 at University of Inland Norway.
Presented as a toturial and introduction to a collaborative scientific workflow using R, RStudio and GitHub.

## Content
### /data
Stores xlsx files used for creating data packages and creating ggplots

### /resources
Stores images, license, bibliography and citation styles.
